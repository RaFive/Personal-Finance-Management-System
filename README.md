# Personal Finance Management System (Sistem Manajemen Keuangan Pribadi)

The **Personal Finance Management System** is a desktop application designed to help users efficiently manage their personal finances. This project was created as a college assignment, showcasing a practical implementation of Java Swing with SQLite database integration.  

---

## Features
- **Login & Sign Up**  
  Securely manage user accounts with a simple authentication system.  

- **Finance Dashboard**  
  View a summary of your finances, including income, expenses, and current balance.  

- **Data Visualization**  
  Visualize your income and expenses using **Donut Charts** for better financial insights.  

- **Transaction Management**  
  Easily add, delete, and track your financial transactions.  

- **Transaction History**  
  Keep a record of all past transactions for long-term financial planning.
